# Chaumian Mix Network
[Mix network](https://en.wikipedia.org/wiki/Mix_network)

## Requirements
Python 3.7+

## Dependencies
- [cryptography](https://pypi.org/project/cryptography/)
- [msgpack](https://pypi.org/project/msgpack/)
- [PyYAML](https://pypi.org/project/PyYAML/)

## Usage

To start a node:

`./node s1`

To start a client:

`./client alice`

Help text:

`./node -h`

`./client -h`
